# Banco QuemPoupaTem 2.0

> Status do projeto: Finalizado

> Este projeto foi desenvolvido durante as férias de julho de 2024.

> Programado no Visual Studio Code.

## :pencil: Descrição do Projeto

A ideia desse projeto era refazer o trabalho de Fundamentos de Algoritmos, realizado no primeiro semestre de Ciência da Computação na FEI, seguindo, de forma geral, as mesmas regras do projeto original. A única diferença é que, neste projeto, acabei esquecendo de desenvolver a parte de senha dos usuários.
